# debian-consul

## [releases](https://github.com/ahdinosaur/debian-consul/releases)

uses [`ahdinosaur/hashicorp-deb`](https://github.com/ahdinosaur/hashicorp-deb) to `download`, `build`, `package`, and `install` Hashicorp projects.
